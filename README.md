# 🧩 בונה הפאזל של המשפט

משחק PWA חינוכי לילדים — מצא את מילת הקישור הנכונה וחבר את שני חלקי המשפט!

## תכונות
- 3 רמות קושי (מתחיל / מתקדם / אלוף)
- 13 חידות עם מילות קישור בעברית
- פידבק מיידי + מנגנון רמזים
- הינשוף מלווה עם הודעות דינמיות
- קונפטי + מסך תוצאות עם כוכבים
- עובד אופליין (Service Worker)
- PWA — ניתן להתקנה על מסך הבית

## קבצים
- `index.html` — מבנה המסכים
- `app.js` — לוגיקת המשחק
- `app.css` — עיצוב RTL מלא
- `puzzles.js` — בסיס נתוני החידות
- `sw.js` — Service Worker
- `manifest.json` — הגדרות PWA
